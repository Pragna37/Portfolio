Idempotency Keys - AI Summary

Built a REST API that solves a real production problem — duplicate network requests. When a user double-clicks a button, the server detects the duplicate using an idempotency key combined with a SHA-256 hash of the request text, returns the cached result from Redis instantly, and skips the AI call entirely. Integrated Groq's LLaMA 3.3 model for real-time text summarization.
